# Golden Eye - Lokator Tambang Ilegal

Golden Eye adalah aplikasi untuk mendeteksi dan melacak lokasi tambang ilegal. Sistem ini membantu pihak berwenang dan masyarakat dalam memantau aktivitas pertambangan yang tidak berizin.

## Instalasi

```bash
git clone https://github.com/username/golden-eye.git
cd golden-eye
bun install
```

## Penggunaan
```bash
bun start
```

## Pengembangan
```bash
bun dev
```
